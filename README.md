# Employee Management App

## Overview
A Python application for managing employee records with MySQL-backed CRUD operations.

## Features
- Add employee records
- Fetch employee details
- Update employee data
- Delete employee records
- Database connection pooling for more efficient access

## Tech Stack
- Python
- Tkinter
- MySQL

## Notes
This project was built to practice database-backed application development and CRUD workflows.

## Setup

1. Create a MySQL database named `employee`
2. Set environment variables:

- `DB_HOST`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`

3. Run the app:

```bash
python employee_crud_app.py
